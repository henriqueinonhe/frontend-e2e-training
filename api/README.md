# Don't Forget

Note taking app API.

## Installation

```sh
npm install
```

## Seeding

```sh
npm run db:seed
```

## Running

```sh
npm run start
```

App uses port 3001 by default, but it can be changed by setting `PORT` environment variable.

```sh
PORT=1234 npm run start
```

## Docs (Swagger)

Docs (Swagger) are located under `/docs`.
